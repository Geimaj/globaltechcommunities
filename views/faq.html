GENERAL

What is Global Tech Communities?

Global Tech Communities is a database of tech communities. Please add your event or tech community, or any that you know of to make this database as complete as possible.

Who created this website?

Robyn Farah from KATO created this site as her aim is to connect people in different tech sectors with those interested in tech in the easiest way possible.

Who can join this site?

Anyone can join this site. In order to be part of the database, you need to be running a tech event and/or community.
